# Sorteador

Olá! Esse é um sorteador, onde você pode inserir valores e sortea-los 

## Features:
* **Sortear:** "Sorteia os valores inseridos"<br>
* **Reiniciar:** "Reinicia os nomes anteriormente inseridos"<br>

## Tecnologias utilizadas:

* [HTML](https://developer.mozilla.org/pt-BR/docs/Web/HTML): Linguagem de hipermarcação de texto
* [CSS](https://developer.mozilla.org/pt-BR/docs/Web/CSS): Linguagem folhas de estilo em cascata
* [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript): linguagem de programação

## Como executar:

### **1. Faça um clone [desse repositório]() na sua máquina:**

* Crie uma pasta no seu computador para esse programa, recomendo colocar o nome **Sorteador**
* Abra o `git bash` ou `terminal` dentro dessa pasta
* Copie a [URL](https://github.com/pirologustavo/sorteador) do repositório
* Digite `git clone <URL copiada>` e pressione `enter`

### **3. Execute o site pelo arquivo index.html**
